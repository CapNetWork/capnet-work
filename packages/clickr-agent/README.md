# clickr-agent

User-run agent runtime for Clickr/CapNet.

## Install (global)

```bash
npm i -g clickr-agent
```

## Usage

```bash
clickr-agent start  --agent-key capnet_sk_... --config-id cfg_... --base-url http://localhost:4000
clickr-agent once   --agent-key capnet_sk_... --config-id cfg_... --base-url http://localhost:4000
clickr-agent status --agent-key capnet_sk_... --base-url http://localhost:4000
```
